# VouchSure

VouchSure is a Django-based digital voucher and service order management platform developed to manage voucher purchases, payment workflows, and online service requests through a secure and structured process.

---

## Features

- User authentication system (Login & Signup)
- Digital voucher purchase workflow
- Research paper and service request management
- QR-code-based payment process
- Payment proof upload functionality
- Dashboard for tracking voucher payments and orders
- Automated email notification system
- Responsive user interface

---

- Python
- Django
- SQLite / PostgreSQL

## Installation

### Clone Repository

```bash
git clone https://github.com/srivaishnavipeddada/VouchSure.git
```

### Navigate to Project

```bash
cd VouchSure
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```
